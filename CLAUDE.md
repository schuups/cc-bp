# Claude Code Blueprint

This repository is a reusable `.claude/` scaffolding for software projects.

Working Agreements (mode selection, roles, gate policy, standing rules) are in `.claude/CLAUDE.md`.

## Quick Reference

| What | Where |
|------|-------|
| Working Agreements | `.claude/CLAUDE.md` |
| Role contracts | `.claude/roles/` |
| Mode protocols | `.claude/protocols/` |
| Pre-commit gate | `.claude/pre-commit.md` |
| Domain model | `specs/domain-model.md` |
| Invariants | `specs/invariants.md` |
| ADRs | `specs/adr/` |
| Fidelity index | `specs/fidelity-index.md` |
| Findings log | `specs/findings.md` |
| Escalations | `specs/escalations.md` |
