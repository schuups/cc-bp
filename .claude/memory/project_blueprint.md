This repo (cc-bp) is the blueprint itself — not a software product. All work here is meta: designing, refactoring, and improving the `.claude/` folder that gets copied into other projects.

**Why:** The user copies this `.claude/` folder into new or existing projects to apply the development framework.

**How to apply:** Keep this context in mind when evaluating scope; changes here affect every future project that uses the blueprint.
